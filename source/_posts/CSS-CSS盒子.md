## CSS盒子：

#### 简单的一个盒子：

```html
<div>
    
</div>
```

```CSS
div{
    width: 100px;
    height: 100px;
    border: 1px solid black;
    border-width: 10px;
}    
```

### 文本的一些CSS属性：

```css
/*文本对齐方式*/
text-align: left;
/*文本所占高度*/h
line-height：200px;
/*首行缩进(缩进几个字体)*/
text-indent: 2em;
1em=1font-size
/*中划线/没有线/underline*/
text-decoration: line-through/none/underline
/*设置光标*/
cursor: pointer/help;
```
